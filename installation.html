Installation

  • Requirements:

    • Node.js
    • A testing framework, such as mocha and chai.

  • Install from the command line:

    $ npm install https://github.com/milewgit/TinyMockJS/raw/master/TinyMockJS-0.2.0.tgz

  • Install via package.json:

    $ cat package.json
{
  "name": "My Project",
  "description": "A description of My Project.",
  "repository": "git://github.com/me/MyProject.git",
  "scripts": {
    "test": "cake test"
  },
  "devDependencies": {
    "mocha": "*",
    "chai": "*",
    "coffee-script": "~1.6",
    "TinyMockJS": "https://github.com/milewgit/TinyMockJS/raw/master/TinyMockJS-0.2.0.tgz"
  }
}

$ npm install

  • Uninstall:

    $ npm uninstall TinyMockJS
