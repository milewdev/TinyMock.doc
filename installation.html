Installation

Requirements:

  • Node.js
  • A testing framework, such as mocha and chai.

Install from the command line:

$ npm install https://github.com/milewgit/TinyMockJS/raw/v0.2.0/TinyMockJS-0.2.0.tgz

Install via package.json:

$ cat package.json
{
  "name": "My Project",
  "description": "A description of My Project.",
  "repository": "git://github.com/me/MyProject.git",
  "scripts": {
    "test": "cake test"
  },
  "devDependencies": {
    "mocha": "^1.18.2",
    "chai": "^1.9.1",
    "coffee-script": "^1.7.1",
    "TinyMockJS": "https://github.com/milewgit/TinyMockJS/raw/v0.2.0/TinyMockJS-0.2.0.tgz"
  }
}

$ npm install

Uninstall:

$ npm uninstall TinyMockJS
